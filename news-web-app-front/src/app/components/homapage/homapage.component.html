The Hawkers Journal

About The Hawkers Journal

The Hawkers Journal is a web application created as part of a dissertation primarily focused on being developed using microservices as a proof of concept but as well as demonstrating the developers own skill.

Overall, the website allows for users to read daily news articles based on news categories but also set their preferences with the creation of an account for a more customised news reading experience. Users with accounts can set their favourite news categories they wish to read regularly from along with the country they wish to receive news from, they can also blacklist certain news sources from their news feed.
